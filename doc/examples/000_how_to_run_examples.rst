========================================================================================================================
How to run examples
========================================================================================================================

This code examples can be run using already prepared tests in

``al-java/doc/code_samples/java/imasjava/examples/TestNewExamples.java``.

1. Change the directory to ``al-java/doc/code_samples/java/``.

.. code-block:: bash

    cd al-java/doc/code_samples/java


2. To **compile** the code, run the following command:

.. code-block:: bash

    javac -cp $CLASSPATH:. imasjava/examples/TestNewExamples.java



3. To **run** the code, use the following command:

.. code-block:: bash

    java -cp $CLASSPATH:. imasjava.examples.TestNewExamples
